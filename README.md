## PTUTime

- <h3>개요</h3>
  - 실시간 대학교 익명 커뮤니티 사이트 제작 
  
- <h3>제작과정</h3>
  - 프로젝트명: <b>PTUTime</b> <br>
  - 수행기간:  2018. 09. 27 ~ 2018. 11. 14 <br>
  - 프로젝트 인원:  3명  <br>
  - 나의 기여도:  50% <br>
  - 담당: 협업 팀장 ,데이터베이스 설계, 메뉴바, 다운로드, 문서 작성 <br>
  
- <h3>개발환경</h3>
  - DB : MySQL(8.0.12) <br>
  - JAVA8 <br>
  - 프레임 워크 : Eclipse Jee Oxygen <br>
  - 서버 : Apache Tomcat(8.5) <br>
  - HTML, CSS, BootStrap, JavaScript <br>
